How to find your Sheet ID
1
Open your Google Sheet in a browser.
2
Look at the URL bar. It looks like:
docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFM…/edit
3
Copy the long ID between /d/ and /edit and paste it above.
Sheet format tips
  • First row must be column headers
  • One product per row
  • Price column should be numbers only (₦ signs OK)
  • Stock column: 999 or blank = unlimited
  • Make sure Phixtra's Google account can view the sheet (or make it public)
