{# Automated Cross-selling (Related Product Cards) — Feature 5 #}

After displaying recommended products, the widget automatically appends up to 2 related products from the same category and price range — labelled "You may also like:". Requires Product Recommendation to also be enabled.

Current status: {% if tenant_features.get('related_products') %} ✓ ACTIVE {% else %} ✗ Inactive {% endif %}

{# Intelligent Cart Revenue Recovery #}

Detects cart abandonment in real time, scores intent (0–100), and automatically runs a 3-touch recovery sequence: a chat widget popup, a personalised AI email (T+2 hr), and a final urgency email (T+24 hr). Targets 65%+ recovery rate vs Tidio's ~35%.

Current status: {% if tenant_features.get('cart_recovery') %} ✓ ACTIVE {% else %} ✗ Inactive {% endif %}

{# Chat Archive — 30 Days #}

Gives the customer access to 30 days of conversation history. Includes fulltext keyword search, PDF export and AI conversation summaries. Excel and Word exports are not included in this tier. Free/trial accounts see only the last 3 days with no search or export.

Current status: {% if tenant_features.get('chat_archive_30days') %} ✓ ACTIVE {% else %} ✗ Inactive {% endif %}

{# Chat Archive — Unlimited #}

Full unlimited conversation history with no day limit. Includes fulltext keyword search, PDF, Excel and Word export and AI conversation summaries. If both 30-Day and Unlimited are ticked, Unlimited takes priority.

Current status: {% if tenant_features.get('chat_archive_unlimited') %} ✓ ACTIVE {% else %} ✗ Inactive {% endif %}

{# WhatsApp Message Templates #}

Enables the Message Templates page in the WhatsApp section of the portal. Lets the merchant map Meta-approved templates to automated events — cart abandonment and order status updates. Requires the WooCommerce plugin to be installed on the merchant's site.

Current status: {% if tenant_features.get('whatsapp_message_templates') %} ✓ ACTIVE {% else %} ✗ Inactive {% endif %}
