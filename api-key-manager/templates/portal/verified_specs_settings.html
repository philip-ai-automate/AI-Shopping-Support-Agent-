⚡ How this works

When a customer asks for a numeric specification (e.g. "How much does this weigh?") and the answer is not in your store's data, PhiXtra searches the web — but only on websites you trust. It never guesses, and it never uses unreliable sources.

Trusted Domains — add the official manufacturer or supplier websites relevant to your products. PhiXtra already includes major tech brands by default; add your own here for any sector (furniture, appliances, sporting goods, etc.).

Custom Spec Types — if your products have specifications that are not standard tech specs (e.g. "Load Capacity", "Thread Count", "Fuel Tank Size"), add them here. PhiXtra will recognise customer questions about them and extract the right value.

Changes here take effect immediately on the next customer question — no restart needed.

Field guide:
Spec name — what the spec is called in answers, e.g. Load Capacity.
Unit — the unit to look for in web results, e.g. kg, litres, rpm.
Trigger keywords — comma-separated words or phrases that tell PhiXtra a customer is asking about this spec. A customer asking "what is the maximum load?" would match the keyword maximum load.
Qualifier — optional note shown with the answer, e.g. "May vary by model."
