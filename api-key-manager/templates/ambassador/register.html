  1. Get a brand new SIM card or a number that has never been used on WhatsApp. This number is just for business.
  2. If it's already on WhatsApp, they must delete that WhatsApp account on the number first.
  3. Keep the SIM active — they'll need to receive a one-time code (OTP) during setup.
  4. Share the number with PhiXtra — we'll connect it to the WhatsApp Business API during onboarding.
💡 MTN, Glo, Airtel, and 9mobile SIMs all work.
